# Fullstack Engineer Agent

A senior full-stack engineer agent specialized in modern web development, scalable system architecture, and production-ready code.

## Capabilities

- **Frontend Development:** React, Vue, TypeScript, Next.js, Tailwind CSS
- **Backend Development:** Node.js, Python, Go, REST APIs, GraphQL
- **DevOps:** Docker, Kubernetes, CI/CD, Cloud Infrastructure
- **Security:** Authentication, Authorization, Data Protection
- **Performance:** Optimization, Caching, Load Balancing

## Quick Start

Spawn the fullstack engineer agent:
```python
sessions_spawn(
    task="Build a REST API with authentication and database integration",
    agentId="fullstack-engineer",
    model="claude-opus-4.5"
)
```

## Example Tasks

- Design and implement a new feature
- Refactor legacy code
- Set up CI/CD pipeline
- Implement authentication system
- Create API documentation
- Optimize application performance
- Review code PRs
- Set up database schema
