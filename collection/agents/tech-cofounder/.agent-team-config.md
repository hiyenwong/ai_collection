# Tech Co-Founder Agent Team Configuration

## OpenClaw (sessions_spawn)

```python
sessions_spawn(
    task="Build a [product description]",
    agentId="tech-cofounder",
    model="claude-sonnet-4.5",  # Good balance of speed and quality
    thinking="high",             # For complex decision making
    runTimeoutSeconds=600,      # 10 minutes per stage
    cleanup="keep"
)
```

## Claude Code (Agent Teams)

已在 Claude Code UI 中启用 **Agent Teams** 功能。

**Agent 配置**：
- **名称**：tech-cofounder-builder
- **角色**：Builder
- **模型**：Claude Sonnet 4.5
- **工作模式**：Phase A → B → C → D

## 使用示例

### 完整工作流程

**Product Owner（你）**：
```
使用 kickoff.md 模板提供完整工作订单
```

**Tech Co-Founder Agent**：
```
Phase A: Plan-to-Build Brief
- 确认范围
- 列出假设
- 提供架构方案
- 等待批准

Phase B: Implement
- 实现阶段性交付
- 提供运行指令
- 等待反馈

Phase C: Polish
- 添加错误处理
- 代码格式化
- 性能检查

Phase D: Handoff
- 生成 README
- 部署文档
- 维护说明
```

### 关键特性

✅ **分阶段交付** - 每个阶段都有明确产出
✅ **决策审批** - 重大决策需要批准
✅ **完整文档** - 包含 README、测试、部署指南
✅ **可维护性** - 代码结构清晰，注释充分
